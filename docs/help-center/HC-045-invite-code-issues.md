# Invite code does not work

**Article ID:** HC-045  
**Audience:** Registry Participants

---

## Check spelling and characters

Re-enter the code carefully.

- Remove any spaces before or after the code.  
- If you copied the code from a message, copy it again to avoid missing characters.  

---

## Authenticate with the identity your host expects

**Registry Owners** sometimes coordinate verbally with one email inbox while the browser session still reflects another Google profile or mailbox. Sign out and sign back in with the account that matches the invitation context.

---

## Link expired, revoked, or typo in URL slug

If you joined from a link, copy the full link again from the original message and try opening it in a new tab.

If it still fails, ask the **Registry Owner** to reshare the invite link or provide the join code again.

---

## Abuse suspensions and fraud holds

Rare safety blocks can prevent joining. If you believe this is a mistake, contact support.
