# Help Center (authors)

Authoring hub for help-center articles surfaced in-app. Customers open **Documentation**, not this file directly.

Route mapping for builds lives in **DOCUMENTATION_PAGES_INDEX** in this folder.

Articles use stable IDs (**HC-###**) in headings. Source files share those IDs in their filenames; keep IDs and titles synchronized when you add or split articles.

Further factual requirements live in the **product requirements document** and **product brief** in this **development** folder.

Risk-oriented guidance on pooled pledges: **Group pledges and sensitive payout information**.

Legal-style summaries customers read live under the **legal** bundle and its authoring readme in this folder.
